## 👤 關於袁惠鈞牧師
袁惠鈞牧師（Rev. Sidney Yuan）致力於以聖經為中心的講道與教導，專注於：
- 系統神學
- 釋經講道
- 中英雙語教會事工
- 跨文化神學溝通

牧師的講道以清晰、紮實、以經文為本著稱，以字面解釋聖經，以聖經解釋聖經, 並致力於讓信徒能夠在真理中扎根、在生活中實踐信仰。
